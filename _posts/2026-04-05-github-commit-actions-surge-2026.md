Kyle Daigle, COO of Github, [responding to an tweet](https://x.com/kdaigle/status/2040164759836778878) with some data about the insane amount of code GH sees as the result of AI:

> There were 1 billion commits in 2025. Now, it's 275 million per week, on pace for 14 billion this year if growth remains linear (spoiler: it won't.)
> 
> GitHub Actions has grown from 500M minutes/week in 2023 to 1B minutes/week in 2025, and now 2.1B minutes so far this week.
